# LangGraphTutorials
This repository has the practice LangGraph files for building ai agencies in LangGraph with python.

The ai model used in test4_main.py was mlx-community/gemma-4-e4b-it-4bit running via mlx local server.

If using Mac (Apple sillicoln):

To setup the server run >>> python -m mlx_lm.server --model mlx-community/gemma-4-e4b-it-4bit <<< in a separate terminal window.

WINDOWS/LINUX server setup:

If using Windows/Linux (CUDA):

To setup the server install llama cpp >>> pip install llama-cpp-python --extra-index-url https://abetlen.github.io/llama-cpp-python/whl/cu121 <<< in a separate terminal window.

Windows/Linux (CPU only):

To setup the server install llama cpp >>> pip install llama-cpp-python <<< in a separate terminal window.
